# Info
UI for info editing users, tables and watching logs.

# Stack
FastApi, Jinja, HTML, CSS, js, Docker, Swagger 3, MongoDb

# Launch
+ git clone https://github.com/Vadsxd/Admin-UI.git
+ cd Admin-UI
+ fill ```.env```
+ ```docker-compose up -d```

# Swagger
http://127.0.0.1:8000/docs
